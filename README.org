* orglib
  a javascript library that provides objects to represent, parsers to
  convert, and writers to persist OrgMode files and content,
  http://orgmode.org/. 

** Installation
   Using npm:
   #+BEGIN_SRC js
   npm i orglib --save
   #+END_SRC

   In Node.js:
   #+BEGIN_SRC js
   var orglib = require('orglib');
   #+END_SRC
